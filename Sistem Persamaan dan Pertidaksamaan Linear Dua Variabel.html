BAB 1: SISTEM PERSAMAAN LINEAR DUA VARIABEL

1.1 Pengertian Persamaan Linear Dua Variabel

Persamaan Linear Dua Variabel (PLDV) adalah sebuah bentuk relasi sama dengan pada bentuk aljabar yang terdiri dari dua variabel, dimana kedua variabelnya berpangkat satu dan pangkat tertingginya hanya satu. Dinamakan persamaan linear karena bentuk grafik persamaan ini merupakan sebuah grafik garis lurus (linear).(Muawwana, 2020)

Perhatikan persamaan-persamaan berikut

  1. x + 6 = y
  2. 2a – b = 4
  3. 3p + 6q = 10

Persamaan-persamaan di atas merupakan contoh bentuk persamaan linear dua variabel. Variabel pada persamaan x + 6 = y adalah x dan y, variabel pada persamaan 2a – b = 4 adalah a dan b. Adapun variabel pada persamaan 3p + 6q = 10 adalah p dan q. Perhatikan bahwa pada setiap contoh persamaan di atas, variabelnya berjumlah dua dan masing-masing variabelnya berpangkat satu (pangkat tertingginya satu).

Perhatikan persamaan x + y = 5. Persamaan x + y = 5 masih merupakan kalimat terbuka. Maksud kalimat di sini berarti persamaan tersebut belum mempunyai nilai kebenaran, karena masih ada nilai yang belum diketahui. Jika nilai x kita ganti dengan bilangan 1, maka nilai y yang memenuhi adalah 4. Karena pasangan bilangan (1, 4) memenuhi persamaan tersebut, maka persamaan x + y = 5 menjadi kalimat yang benar. Dalam hal ini, dapat dikatakan bahwa (1, 4) merupakan salah satu solusi penyelesaian dari persamaan x + y = 5. (Nuharini & Wahyuni, 2008)

Untuk mencari nilai x dan y yang memenuhi persamaan x + y = 5 akan lebih mudah dengan membuat tabel seperti berikut.

Rene Descartes lahir pada 31 Maret 1596 di sebuah desa di Prancis. Dia menempuh pendidikan di Belanda, ia juga suka belajar matematika di waktu luang. Karya Descartes yang paling dikenal di antaranya adalah geometri Cartesian yang menggunakan aljabar untuk menggambarkan geometri. Kemungkinan Descartes menemukan istilah untuk “Sistem Persamaan Linier (Linear Equation)” yaitu saat dia belajar di Belanda. Eliminasi Gauss merupakan salah satu solusi dalam sistem persamaan linear.

Metode di Eropa berasal dari catatan Isaac Newton. Pada tahun 1670, dia menulis bahwa seluruh buku aljabar yang diketahuinya memiliki kekurangan solusi untuk memecahkan persamaan simultan. Carl Friedrich Gauss pada tahun 1810 menyusun notasi untuk eliminasi simetrik yang diadopsi pada abad ke-19 oleh komputer tangan profesional untuk memecahkan persamaan normal masalah kuadrat hingga terkecil. Metode eliminasi Gauss dinilai kurang efisien untuk menyelesaikan sebuah SPL. Namun, pada perkembangannya kini, metode eliminasi Gauss disempurnakan menjadi eliminasi Gauss-Jordan. Motode tersebut dinamai Eliminasi Gauss-Jordan untuk menghormati Carl Friedrich Gauss dan Whilhelm Jordan. Wilhelm Jordan (1842-1899) merupakan seorang insinyur asal Jerman yang ahli dalam bidang geodesi. Buku karya nya tentang sistem persamaan linear berjudul Handbuch de Vermessungskunde (Buku panduan Geodesi) pada tahun 1988. (Nurlianawati, 2020)



1.3 Pengertian Sistem Persamaan Linear Dua Variabel

Sistem persamaan linear dua variabel (SPLDV) adalah sebuah sistem / gabungan / kesatuan dari beberapa Persamaan Linear Dua Variabel yang sejenis (misal variabel x dan y). Jadi, sebelum mempelajari Sistem Persamaan Linear Dua Variabel (SPLDV), kita harus terlebih dahulu mempelajari mengenai hal – hal yang berhubungan dengan Sistem Persamaan Linear Dua Variabel (SPLDV) (Siska, 2014).(Muawwana, 2020)

Dengan kata lain, Sistem Persamaan Linear Dua Variabel (SPLDV) adalah beberapa persamaan linear yang masing-masing persamaannya terdiri dari dua variabel (misal x dan y) yang koefisiennya a dan b serta konstanta misalnya c. (Nurlianawati, 2020). Jadi, sistem persamaan linear dua variabel merupakan kumpulan dari dua atau lebih persamaan-persamaan linear yang terdiri dari dua variabel yang sejenis. Pada persamaan hanya terdapat sebuah persamaan saja, sedangkan pada sistem persamaan terdapat dua atau lebih persamaan yang variabelnya sama dan saling berkaitan satu sama lainnya, dimana dua variabel yang ada menunjukan banyaknya variabel yang akan ditentukan penyelesaiannya.


Bentuk umum dari sistem persamaan linear dua variabel adalah sebagai berikut:

a1x + b1y = c1 ; a2 x + b2y = c2

dengan a, b, dan c ∈ R
a, b ≠ 0
x dan y merupakan variabel





1.4 Bagian-bagian Sistem Persamaan Linear Dua Variabel

Dalam sistem persamaan linear dua variabel, terdapat bagian-bagian atau istilah-istilah yang ada di dalamnya. Berikut bagian-bagian pada sistem persamaan linear dua variabel, yaitu:

1. Variabel
Suatu pemisal / pengganti dari suatu nilai atau bilangan, yang biasa dilambangkan dengan huruf atau simbol disebut dengan variabel.
Contoh: Sinta memiliki 5 ekor kuda dan 2 ekor kucing

Jika ditulis dengan memisalkan: x = kuda dan y = kucing,
Maka menjadi: 5x + 2y, dimana x dan y adalah variabel.

Gambar 8. Variabel pada PLDV


2. Koefisien
Ada sebuah bilangan yang menyatakan banyaknya jumlah variabel yang sejenis, disebut dengan koefisien. Jadi, koefisien merupakan bilangan yang biasanya berada di depan variabel dimana bilangan itu menunjukkan jumlah variabel yang sejenis.
Contoh: 3x + 4y

Maka angka 3 dan 4 merupakan koefisien (3 adalah koefisien x, sedangkan 4 adalah koefisien y).

Gambar 9. Koefisien pada Persamaan Linear


3. Konstanta
Suatu nilai atau bilangan yang tidak diikuti oleh variabel, disebut dengan konstanta. Jadi, nilai konstanta tidak bergantung pada nilai variabelnya. Dengan kata lain, berapapun nilai variabel, konstanta akan selalu tetap (konstan).
Contoh: 3x + 4y = 20

Dari persamaan di atas, diketahui bahwa angka 20 merupakan suatu konstanta. Hal itu dikarenakan berapapun nilai x dan y nya, nilai 20 tidak akan terpengaruh, melainkan akan selalu tetap.

Gambar 10. Konstanta pada PLDV


4. Suku
Pada persamaan, terdapat bagian-bagian dari bentuk aljabar yang terdiri dari variabel, koefisien, ataupun konstanta yang dipisahkan oleh tanda operasi penjumlahan. Bagian-bagian tersebut disebut dengan suku.
Contoh: 5x- y + 8, maka suku-sukunya adalah: 5x, -y, dan 8 (berjumlah 3 suku)

1.5 Metode Penyelesaian Sistem Persamaan Linear Dua Variabel

Ada beberapa metode dalam menyelesaikan sistem persamaan linear dua variabel, yakni:

  1. Grafik
  2. Eliminasi
  3. Substitusi
  4. Gabungan (eliminasi dan substitusi)


  1. Metode Grafik

    Penyelesaian SPLDV dengan metode grafik adalah melalui koordinat titik potong grafik dari garis persamaan-persamaan linearnya.

  2. Metode Eliminasi

    Pada metode eliminasi ini, cara untuk menentukan himpunan penyelesaian dari sistem persamaan linear dua variabel ialah dengan menghilangkan (mengeliminasi) salah satu variabel yang ada pada sistem persamaan tersebut (Arifin, 2020)(Muawwana, 2020).
    Dalam metode eliminasi, salah satu variabelnya akan dieliminasi atau dihilangkan menggunakan operasi aljabar, yaitu dengan cara mengurangkan kedua persamaan yang ada. Agar variabel yang diinginkan tereliminasi pada saat dikurangkan, maka terlebih dahulu kita harus menyamakan koefisien variabel tersebut. (Simangunsong, 2016). Misal variabelnya yaitu x dan y, maka untuk menentukan variabel x kita harus mengeliminasi variabel y terlebih dahulu, dan sebaliknya. (Muawwana, 2020)

  3. Metode Substitusi

    Metode Substitusi adalah suatu metode yang digunakan untuk menyelesaikan sebuah sistem persamaan linear dua variabel, dimana terlebih dahulu kita harus menyatakan suatu variabel ke dalam variabel yang lain dari suatu persamaan, lalu kita mensubstitusikan (menggantikan) variabel itu dalam persamaan yang lainnya. (Muawwana, 2020). Dalam metode ini, salah satu variabel dipisahkan dari satu persamaan yang ada, seperti: y = mx + k atau x = my + k, lalu variabel yang sudah dipisahkan ini disubstitusikan ke persamaan yang lainnya. (Simangunsong, 2016)

  4. Metode Gabungan

    Metode gabungan adalah suatu metode penyelesaian SPLDV, dimana kita menggabungkan atau memakai dua metode, yakni metode eliminasi dan substitusi. (Muawwana, 2020)

1.7 Jenis Penyelesaian Sistem Persamaan Linear

Secara umum, ada tidaknya penyelesaian sistem persamaan linear

  • ax + by = c
  • dx + ey = f

dapat dirumuskan seperti berikut ini:

  1. Sistem persamaan linear mempunyai penyelesaian yang unik jika (-a)/b ≠ (-d)/e atau jika a/d ≠ b/e
  2. Sistem persamaan linear tidak mempunyai penyelesaian jika a/d = b/e
  3. Sistem persamaan mempunyai tak hingga penyelesaian jika a/d = b/e = c/f





1.8 Beberapa Sistem Persamaan yang Dapat Diubah Menjadi Sistem Persamaan Linear

Sistem persamaan berbentuk

  • a/x + b/y = c
  • d/x + e/y = f

dapat diubah menjadi
sistem persamaan linear dengan memisalkan 1/x = p dan 1/y = q

Demikian juga sistem persamaan

  • a/(x+y) + b/(x-y) = c
  • d/(x+y) + e/(x-y) = f

Dapat diubah menjadi
sistem persamaan linear dengan memisalkan 1/(x+y) = p dan 1/(x-y) = q.

Pemisalan tersebut menghasilkan sistem persamaan linear dengan variabel p dan q. (Simangunsong, 2016)

2.1 Pengertian Pertidaksamaan Linear Dua Variabel

Pertidaksamaan linear dua variabel adalah kalimat terbuka yang menggunakan tanda pertidaksamaan, seperti >, <, ≤ , atau ≥, dan mengandung dua variabel yang berpangkat satu.

Pertidaksamaan dengan bentuk:

ax + by ≥ c; ax + by > c; ax + by ≤; ax + by < c

2.2 Pengertian Sistem Pertidaksamaan Linear Dua Variabel

Sistem pertidaksamaan linear dua variabel adalah dua atau lebih pertidaksamaan linear dua variabel, yang dimana antara pertidaksamaan yang satu dengan yang lain saling berkaitan.

Gambar 21. Grafik Sistem Pertidaksamaan Linear Dua Variabel


Bentuk umum dari sistem pertidaksamaan linear dua variabel:
ax + by > c; ax + by ≥ c; ax + by < c; ax +by ≤ c dengan a, b, c ∈ R





2.3 Sifat-sifat Pertidaksamaan

Sebelum memahami sistem pertidaksamaan linear, kita harus terlebih dahulu memahami sifat-sifat dari pertidaksamaan itu sendiri.

  1. Sifat tak negatif Untuk a ∈ R maka a ≥ 0.


  2. Sifat transitif
    3. Untuk a, b, c ∈ R jika a < b dan b < c maka a < c;

    jika a > b dan b > c maka a > c

  3. Sifat penjumlahan
    4. Untuk a, b, c ∈ R jika a < b maka a + c < b + c
    jika a > b maka a + c > b + c
    Jika kedua ruas pertidaksamaan dijumlahkan dengan bilangan yang sama, tidak mengubah tanda ketidaksamaan.

  4. Sifat perkalian
    5. Jika a < b, c > 0 maka ac < bc
    Jika a > b, c > 0 maka ac > bc
    Jika a < b, c < 0 maka ac < bc

    Jika kedua ruas pertidaksamaan dikalikan bilangan (riil) positif tidak akan mengubah tanda ketidaksamaan, sedangkan jika dikalikan bilangan negatif akan mengubah tanda ketidaksamaan.

  5. Sifat kebalikan
    Jika a > 0 maka 1/a> 0
    Jika a < 0 maka 1/a < 0




2.4 Himpunan Penyelesaian Pertidaksamaan Linear Dua Variabel

Himpunan penyelesaian dari pertidaksamaan dapat digambarkan pada garis bilangan, khususnya untuk himpunan penyelesaian berupa interval. Batas-batas interval digambarkan dengan menggunakan tanda (bulatan penuh) atau (bulatan kosong). Tanda bulatan penuh menunjukkan bilangan tersebut termasuk ke dalam himpunan penyelesaian, dan sebaliknya tanda bulatan kosong menunjukkan bilangan tersebut tidak termasuk ke dalam himpunan penyelesaian. (Gumilar, 2008, p. 69)
